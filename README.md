# 🍽️ The Flavour Feed

> **Discover delicious recipes from around the world, one dish at a time! 🌍**

The Flavour Feed is a beginner-friendly recipe website built using **HTML, CSS, and JavaScript**. It allows users to explore popular dishes from different countries and view detailed recipes, ingredients, cooking time, and step-by-step cooking instructions.

---

## ✨ Features

🌎 Explore recipes from **5 countries**

- 🇮🇳 India
- 🇰🇷 South Korea
- 🇺🇸 America
- 🇨🇳 China
- 🇮🇹 Italy

🍛 Each country includes

- 4 Popular Dishes
- Dish Image
- Cooking Time
- Ingredients List
- Step-by-Step Recipe

🎨 User-Friendly Interface

- Responsive Design
- Clean Card Layout
- Interactive Country Selection
- Smooth Scrolling
- Hover Effects
